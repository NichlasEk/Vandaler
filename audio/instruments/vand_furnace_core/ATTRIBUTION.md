# Vand Furnace Core Instrument Bank

This bank contains a small curated subset of Furnace instrument data converted
to Vandaler's neutral `vandaler-instrument-v0` JSON format.

Source project:

- Furnace by tildearrow and contributors: https://github.com/tildearrow/furnace
- Source instrument folders:
  - https://github.com/tildearrow/furnace/tree/master/instruments/OPN
  - https://github.com/tildearrow/furnace/tree/master/instruments/SN7
- Instrument contribution policy:
  - https://raw.githubusercontent.com/tildearrow/furnace/master/instruments/README.md

The Furnace runtime/code is GPL and is not vendored here. These files are
converted instrument data only. The bank metadata keeps the original raw source
URL for each imported `.fui` file.

Curated files:

- `bass_growl_00_wobbly.fui`
- `fm_bass1.fui`
- `fm_grinder.fui`
- `growl.fui`
- `Echo.fui`
- `Octave Arp.fui`
